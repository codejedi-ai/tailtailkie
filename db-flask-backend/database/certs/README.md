# These certificates are for local development ONLY

The certificates in this directory are generated by executing
```
./database/generate_certs.sh
```
script from the root of the project.

These are produced by `mysql_ssl_rsa_setup` binary
(see [mysql_ssl_rsa_setup — Create SSL/RSA Files](https://dev.mysql.com/doc/refman/8.0/en/mysql-ssl-rsa-setup.html)
for more details) of the local MySQL instance, which is never deployed
into any of our GKE environments.
